- Javascript 引擎

  - SpiderMonkey: Brendan Eich 開發
  - Chakra: 微軟開發，用於 IE
  - Javascript Core: Webkit 中的 JS 引擎，Apple 公司開發
  - V8: Google

- 瀏覽器內河與引擎的關係
  - WebCore: 負責 HTML 解析、布局、選染等相關工作。
  - JavaScriptCore: 解析、 執行 JS 程式
  - 舉例:
    - IOS UIWebView
    - Android Webview
    - wxml wxss
